# 🎬 CineBox

CineBox é um projeto web inspirado em plataformas como **Letterboxd**, desenvolvido como **projeto acadêmico**.  
A aplicação tem como objetivo permitir que usuários descubram filmes, visualizem informações detalhadas e compartilhem avaliações de forma simples e intuitiva.

---

## Funcionalidades

- **Home com gêneros de filmes**  
  Menu interativo onde o usuário pode selecionar gêneros (Ação, Comédia, Ficção, Romance, Animações, etc).

- **Listagem de filmes por gênero**  
  Ao clicar em um gênero, o usuário é direcionado para uma nova tela com os **filmes do momento** relacionados à categoria escolhida.

- **Detalhes do filme**  
  Cada filme possui:
  - Card individual
  - Sinopse
  - Área de comentários para avaliação dos usuários

- **Sistema de comentários e avaliação**  
  Os usuários podem deixar comentários sobre os filmes, simulando uma experiência de avaliação semelhante a plataformas reais.

- **Tela de Login e Cadastro**  
  Sistema simples de autenticação para controle de acesso às funcionalidades.

---

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla / JS puro)**
- **LocalStorage** (armazenamento local de dados)

> ⚠️ Atualmente, o projeto **não utiliza banco de dados**, devido às limitações de tempo.  
> Os dados de login, cadastro e comentários são armazenados no **LocalStorage**.
